# Fishjam Chat

This is an example app demonstrating how to build a simple conferencing react app using Fishjam.

Install dependencies with `npm install`, run with `npm run dev`.

## Environment Setup

1. Create a `.env` file in the project root.
2. Add your Fishjam ID:

```bash
VITE_FISHJAM_ID=your_fishjam_id
```
